# 달란트마블

휴대폰과 PC에서 함께 접속하는 교회용 실시간 보드게임 첫 버전입니다.

## 실행

Windows PowerShell에서 이 폴더를 열고 다음을 실행합니다.

```powershell
node server.js
```

같은 컴퓨터에서는 `http://localhost:3000`으로 접속합니다. 같은 Wi-Fi의 휴대폰에서는 교사용 PC의 로컬 IP 주소를 확인해 `http://PC의-IP:3000`으로 접속합니다.

## 포함된 기능

- 방 생성과 2-6팀 참여
- 실시간 상태 동기화(1초 간격)
- 주사위, 이동, 출발 보상, 더블 추가 턴
- 성경 땅 구입, 건물, 통행료
- 황금열쇠와 성경 퀴즈
- 광야·세계여행 칸의 기본 보드 구성

게임 서버를 종료하면 현재 게임 방은 초기화됩니다. 다음 단계에서는 영구 저장, 관리자 규칙 설정, 빔프로젝터 관전 화면, QR 입장 기능을 추가할 수 있습니다.
